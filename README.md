# Aof-code

เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์เพื่อดูเว็บไซต์เวอร์ชัน demo ที่ทำงานได้แบบออฟไลน์

ในเว็บไซต์นี้มีระบบจำลอง AI 4 ตัว (Gemini, Meta Llama, DeepSeek, Qwen) ที่ตอบสนองคำสั่ง `/plan`, `/gencode`, `/chat`, `/review`, `/fix`, `/export`, `/agents` และคำสั่งทั่วไป

ฟีเจอร์เพิ่มเติม:
- ระบบล็อกอินด้วยบัญชี local browser
- จำกัดคำสั่งต่อช่วงเวลา (rate limit)
- ประวัติและสถานะเซสชันเก็บใน localStorage
- เลือก tech stack ก่อน /gencode (React / Vue / Next.js + Node / Python / Go)
- เสนอไฟล์ที่สร้างเป็นแท็บหลายไฟล์และดาวน์โหลดเป็น `.zip`
- ฟีเจอร์ห้องร่วมงานเบื้องต้น (room) สำหรับแชร์โครงการในแท็บเดียวกัน
